## Dependencies

This project uses:
- React 18.3.1
- React Router v6
- React Scripts 5.0.1
- Sass (dart-sass)

## Building for Production

To build the application for staging:
```bash
npm run build:staging
```

To build the application for production:
```bash
npm run build:production
```
